# ta-te-ti
Proyecto de la materia seminario de lenguajes -  Facultad de Informática- UNLP

# Tecnologías

# Para clonar el repositorio
git clone https://github.com/DiegoHaroVivanco/ta-te-ti.git

# Para correr la app

Ingresar a http://localhost:8080/
